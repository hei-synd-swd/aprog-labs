# Rectangle Struct

Define a struct `Rectangle` with two fields:

- `width` — a `u32`
- `height` — a `u32`

Then, in `main`, create an instance of `Rectangle` with width `10` and height
`20`.

## Expected Result

The program compiles and prints the rectangle's dimensions.
