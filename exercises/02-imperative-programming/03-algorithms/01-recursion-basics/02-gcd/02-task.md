# GCD

Write a recursive function `gcd(a: u64, b: u64) -> u64` that returns the
greatest common divisor using Euclid's algorithm.

Expected Result:
```
gcd(48, 18) → 6
gcd(7, 3)   → 1
gcd(0, 5)   → 5
```
